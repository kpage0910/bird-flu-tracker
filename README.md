# Bird Flu Tracker & Survival Predictor

## Project Goal
Track and visualize global bird flu outbreaks using web-scraped health reports, and simulate a survival dataset to predict patient outcomes using machine learning.

## Why This Matters
Bird flu outbreaks are an ongoing global health concern. While centralized datasets are often unavailable or outdated, this project scrapes real-time information to build a live tracker and uses synthetic modeling to predict outcomes for educational and analytical purposes.

## Project Phases
1. **Scraping**: Extract bird flu outbreak reports from WHO or CDC websites using Python.
2. **EDA & Visualization**: Analyze trends by region, species, strain, and date using Pandas & Plotly/Seaborn.
3. **Simulation**: Generate synthetic patient-level data with factors like age, exposure type, and symptoms.
4. **Prediction**: Train a machine learning model (Logistic Regression or Decision Tree) to estimate survival.
5. **Presentation**: Package as a GitHub portfolio project with notebooks, visuals, and a writeup.
